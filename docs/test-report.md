## 4. 已知限制 / 後續

- 無 refresh token:JWT 過期後,下一次請求會收到 401 並導回登入。
- 後端 CORS 目前反射任何來源,建議上線後以 `CORS_ORIGINS` 收斂為特定網域。
- 分類提供兩個維度:裝置**狀態**(線上/離線/維護)與自訂**類別**(category)。
- 測試為情境式(端到端行為驗證);單元測試覆蓋率擴充列為後續工作。
